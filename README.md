# IntellMeet

IntellMeet is an AI-powered enterprise meeting and collaboration platform.

## Repository structure

- `Frontend/` — React + TypeScript frontend application
- `Backend/` — Express API backend
- `docs/` — product blueprints and architecture documentation
- `docker-compose.yml` — local orchestration entrypoint

## Documentation

- `docs/INTELLMEET_MASTER_BLUEPRINT.md` — master product architecture, feature map, routes, and flows
